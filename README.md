## 搬瓦工是什么？一句话先说清楚

搬瓦工卖的是 KVM 架构的 VPS（虚拟私人服务器），自己拥有硬件和 IP，自建 KiwiVM 控制面板，走的是自托管模式——你买回去自己管，搬瓦工不帮你调 WordPress、不帮你配 Nginx，出了硬件和网络问题他们管，软件问题自己搞。这也是为什么价格能压得相对下来。

对中国大陆用户来说，选搬瓦工的核心理由只有一个：**CN2 GIA 线路**。这是中国电信最高等级的国际专线，晚高峰时段丢包率和延迟都比普通线路稳定得多。

全球目前 25+ 个机房，支持支付宝和银联付款。30 天内不满意可以全额退款，直接后台申请自动处理。

## 三个套餐各买哪个？看场景说话

**新手、学生、低频使用：KVM 套餐 $49.99/年**

$49.99 一年，算下来一天一块多人民币都不到。跑 Linux 练习、部署个小脚本、搭个测试环境，完全够用。线路不是 CN2，晚高峰会有些波动，但价格就是这样，别期待太高。

说实话，很多人买了 KVM 套餐用了几个月，觉得速度还行就继续续，觉得线路不稳就往上换 CN2 GIA-E——搬瓦工本来也支持这个升级路径，差价可以补。

**建站、日常高频、看重速度：CN2 GIA-E 套餐**

这是搬瓦工的主力套餐，也是大多数人的推荐。

CN2 GIA-E 的 E 代表 ECOMMERCE，本来就是搬瓦工为建站场景设计的。三网回程都走 CN2 GIA，晚高峰电信、联通、移动用户基本都能稳住，不会出现白天快晚上卡的情况。

最大的优势是机房切换。买了之后在 KiwiVM 面板里一键操作，12 个以上机房随便迁，不用重装系统，数据保留。今天觉得 DC6 延迟高，明天迁到 DC9 试试，零成本试错。

季付 $49.99 也是对新用户友好的设计——不用一口气付一年，试三个月觉得不合适还有 30 天退款兜底。

**对延迟有极致要求、公费报销：香港/东京套餐**

$89.99 一个月起，这个预算买 CN2 GIA-E 可以用两年了。

但香港套餐的优势是真的无可替代：物理距离近，广东方向延迟能低到 10ms 以内，这个数字美国机房根本达不到。如果你在跑对延迟极度敏感的业务，比如实时游戏加速、企业专线、高频金融交易类应用，香港套餐才是值得的。

普通个人用户，CN2 GIA-E 已经够了，没必要为香港套餐多出那么多钱。

## 常见问题 FAQ

**Q：搬瓦工的套餐续费会涨价吗？**

不会。搬瓦工的设计是续费价格和购买价格一致，包括使用优惠码购买的情况——只要购买时用了优惠码，每次续费都按折扣价算，不是只有首次优惠。

**Q：支持支付宝吗？**

支持。除支付宝外，还支持 PayPal 和银联，国内用户付款基本没障碍。

**Q：IP 被封了怎么办？**

两个选择：在 KiwiVM 付费换 IP（$8 一次），或者迁移到其他机房（CN2 GIA-E 套餐支持）。迁机房需要满足流量使用小于 10% 等条件，具体在后台查看。30 天退款保证同样要求流量使用小于 10%。

**Q：KVM 套餐和 CN2 GIA-E 套餐的线路差距有多大？**

差距比较明显。KVM 套餐走普通国际线路（163/普通直连），晚高峰可能卡；CN2 GIA-E 走中国电信顶级专线，三网都有高端线路保障。如果你的用途对速度有要求，$49.99/年 KVM 和 $49.99/季 CN2 GIA-E 选哪个，基本不用犹豫。

**Q：限量版套餐有没有推荐？**

有货的时候，CN2 GIA-E 系列传家宝和 THE PLAN 系列是最有性价比的。买到之后不要轻易退，传家宝的意思就是长期续费比常规套餐便宜得多。

**Q：搬瓦工适合完全不懂 Linux 的新手吗？**

适合「愿意折腾」的新手，不适合「完全不想学」的人。搬瓦工是自托管 VPS，客服处理硬件和网络问题，应用层面全靠自己。好处是 KiwiVM 面板操作简单，SSH 登录之后基本跟着教程走就能搞定大部分操作。
